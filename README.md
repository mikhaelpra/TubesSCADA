# Student SCADA Baseline Repo v3

Versi ini ditujukan untuk proyek mahasiswa 6 minggu.

## Penandaan file
- `[REUSE]` : boleh digunakan langsung
- `[MODIFY]` : boleh digunakan, tetapi wajib dimodifikasi signifikan
- `[STUDENT]` : wajib dikerjakan mahasiswa

## Target proyek
Membangun operator loop end-to-end:
device/simulator -> gateway/RTU -> SCADA core -> GUI -> command -> feedback -> alarm -> SOE -> trend

## Aturan akademik
1. Semua file `[STUDENT]` wajib diisi mahasiswa.
2. Semua file `[MODIFY]` wajib menunjukkan perubahan nyata dari baseline.
3. File `[REUSE]` boleh dipakai sebagai fondasi, tetapi mahasiswa harus memahami fungsinya.

## Roadmap 6 minggu
- Minggu 1: akuisisi data dan point update
- Minggu 2: command loop
- Minggu 3: alarm dan SOE
- Minggu 4: GUI operator core
- Minggu 5: historian dan trend
- Minggu 6: integrasi, pengujian, dan demo
